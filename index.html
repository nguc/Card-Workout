Card Workout

Ace - Rest 2 minutes!

Spades - Push

  1. Push up or incline pushups
  2. Pike push ups or vertical pushups
  3. Standing dumbbell press
  4. Lateral raises or timed doorway later holds (eg. 16 reps = 16 sec)

Clubs - Pull

  1. Pull ups
  2. Inverted row or dumbbell row
  3. Rear delt flyes
  4. Upright row

Hearts - Legs

  1. Walking lunges
  2. Bulgarian split squat
  3. Single leg hip thrust
  4. Nordic ham curl

Diamonds - Isolation

  1. Biceps: weighted curl or weightless flex curls
  2. Triceps: Single arm side push up or skullcrushers
  3. Abs: Bicycle crunch or reverse crunch
  4. Calves: Standing calf raise
